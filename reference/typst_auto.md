# Typst auto

The `auto` keyword in Typst.

## Usage

``` r
auto
```

## Format

An object of class `typst_auto` (inherits from `typst_expression`) of
length 1.

## Value

An object of class `typst_auto` and `typst_expression`.

## Examples

``` r
auto
#> [1] "auto"
#> attr(,"class")
#> [1] "typst_auto"       "typst_expression"
```
