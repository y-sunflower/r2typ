# Expression like classes/

Class names of expressions like.

Function where positional arguments are in brackets.

Function where additional arguments is not content. For example, image()
usage must be image(height: 2in, "file.svg").

## Usage

``` r
expression_like

pos_bracket_functions

no_bracket_functions
```

## Format

An object of class `character` of length 4.

An object of class `character` of length 3.

An object of class `character` of length 33.
