# Compile a `.typ` file to a `.pdf` file and return the output path.

Compile a `.typ` file to a `.pdf` file and return the output path.

## Usage

``` r
typst_compile(file, output = NULL)
```

## Arguments

- file:

  Path to an existing `.typ` file.

- output:

  Optional output path. Defaults to the input path with `.pdf`.
